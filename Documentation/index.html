On Scroll Animation

you will get some code like this on animated sections 

class="wow fadeInRight" data-wow-duration="2s" data-wow-delay="0s"

"wow" is a common class if you want to animate any section or element then you need to add a class named "wow".

"fadeInRight" this is the animation class . you can find all class name from here http://daneden.github.io/animate.css/ Just Replace those class

(data-wow-duration="2s") you can change animation duration from here

(data-wow-delay="0s") you can change animation delay from here
